StockOps Dev AWS Architecture

Terraform으로 배포된 dev 환경 기준 구조입니다. 버튼을 누르면 관리자 로그인, 센서 메시지, 배포 흐름을 각각 강조합니다.

전체 구조

관리자 웹은 CloudFront와 S3에서 정적 파일로 제공되고, API는 Elastic Beanstalk의 Amazon Linux 2023 Docker 환경에서 동작합니다. RDS PostgreSQL은 애플리케이션 데이터를 저장하고, MQTT와 Sensimul은 ECS Fargate에서 dev 테스트용으로 실행됩니다.

운영 전 보완 포인트

  • API HTTPS 연결 또는 CloudFront/API Gateway 앞단 구성이 필요합니다.
  • RDS와 애플리케이션 계층은 private subnet 중심으로 정리하는 것이 좋습니다.
  • GitLab CI에서 image push, Terraform apply, S3 sync를 분리하면 로컬 수동 작업을 줄일 수 있습니다.
